# Description
Appfood is an App coded in React Native using the StarWars API.

## Build Setup

``` bash
# clone project
$ git clone git@github.com:marioposada/react-native-starwars-app.git

# install dependencies
$ npm install

# serve with host at localhost:19000
$ npm start
$ yarn start
```
